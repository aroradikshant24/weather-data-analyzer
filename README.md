# Weather Data Analyzer

This project analyzes 30 days of weather data including:
- Temperature
- Humidity
- Rainfall

## How to Run
1. Compile the program:
   `gcc src/main.c -o weather`
2. Run:
   `./weather`

## Folders
- src/ — source code  
- include/ — header files  
- docs/ — report  
- assets/ — screenshots  
- sample_input.txt — dataset
